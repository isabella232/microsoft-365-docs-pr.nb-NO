# <a name="microsoft-365-business-frequently-asked-questions"></a>Vanlige spørsmål om Microsoft 365 Business

## <a name="general"></a>Generelt

### <a name="what-is-microsoft-365-business"></a>Hva er Microsoft 365 Business? 
Microsoft 365 er en integrert løsning som samler de beste produktivitetsverktøyene, sikkerheten og enhets administrasjonsfunksjonene for små til mellom store bedrifter. 

**Et sett med verktøy for forretningsproduktivitet og samarbeid** 
* Word, Excel, PowerPoint, Outlook, OneNote, Publisher og Access 
* Exchange, OneDrive, Skype for bedrifter, Microsoft Teams, SharePoint 
* Forretningsprogrammer fra Office (bestillinger, Outlook Customer Manager, MileIQ<sup>[1](#footnote1)</sup>, Microsoft fakturering<sup>[1](#footnote1)</sup>) 
 
**Enhetsbehandling og sikkerhetsfunksjoner i foretaksklassen** 
- Bidrar til beskyttelse mot eksterne trusler som phishing & sofistikert skadelig programvare med Office 365 avansert trusselbeskyttelse og Windows Defender utnytte beskyttelse 
-  Hjelper til med å kontrollere & administrere hvordan sensitiv informasjon åpnes og overføres med Office 365-policyer for hindring av datatap (for å være tilgjengelig om sommeren, 2018) og Azure Information Protection Plan 1 
-  Bidrar til å beskytte, bevare og sikkerhetskopiere data med Exchange Online-arkivering  
-   App beskyttelse for Office & andre Mobilapper med Intune app Protection 
-  Enhetsbehandling for Windows 10-PCer, MacOS & Mobiles enheter med Enhetsbehandling for Intune 
-  Konsekvent sikkerhetskonfigurasjon på tvers av enheter – beskyttelse av firmadata på tvers av enheter-Windows Defender, alltid på og oppdatert 
 
 
**Forenklet enhets distribusjon og Brukeroppsett** 
* Enkel Administrasjonskonsoll for konfigurering og administrering av brukere og enheter 
* Automatisk installasjon av Office-apper på Windows 10-PCer 
* Alltid oppdatert Office + Windows 10 
* Strømlinjeformet distribusjon av PC-er med Windows AutoPilot

**Aktivere betinget tilgang**
- Microsoft 365 bedriftskunder kan aktivere betinget tilgang via Azure Directory-innstillingene i Azure-portalen. Hvis du vil ha mer informasjon om hvordan du konfigurerer policyer for betinget tilgang, kan du se artikkelen [Hva er betinget tilgang](https://docs.microsoft.com/azure/active-directory/conditional-access/overview).

### <a name="who-should-consider-adopting-microsoft-365-business"></a>Hvem bør vurdere å vedta Microsoft 365 Business? 
Microsoft 365 Business ble bygget for små og mellom store kunder som har lite eller ingen IT-ressurser på staben, og som ønsker de beste produktivitets-og samarbeidsfunksjonene i Office 365 sammen med enhetsadministrasjon og sikkerhetsløsninger som ivareta forretningsdata. Microsoft 365 Business-kunden er klar til å flytte sine IT-operasjoner til skyen og er interessert i å opprettholde en proaktiv holdning for å beskytte data på både bedrifts-og ansatt-eide enheter.

### <a name="how-can-i-get-microsoft-365-business-for-my-business"></a>Hvordan kan jeg få Microsoft 365 Business for min bedrift? 
Microsoft 365 Business kan kjøpes via en <a href="https://www.microsoft.com/solution-providers/search" target="_blank">Microsoft-partner</a> eller direkte fra <a href="https://www.microsoft.com//microsoft-365/business" target="_blank">Microsoft</a>. Når du velger om du skal kjøpe direkte fra Microsoft eller via en Microsoft-partner, bør du vurdere om du har mulighet til å opprettholde en IT-infrastruktur. En Microsoft-partner kan hjelpe deg med å distribuere og administrere IT-infrastrukturen, inkludert Microsoft-løsninger.

### <a name="how-much-does-microsoft-365-business-cost"></a>Hvor mye koster Microsoft 365 Business? 
Microsoft 365 Business tilbys for USD $20,00 bruker/måned basert på en _årlig kontrakt_ hvis kjøpt direkte fra Microsoft. Når de er kjøpt via en Microsoft-partner, kan prisene variere basert på tjenestene partneren tilbyr, og prismodellen deres for Microsoft 365 Business. 

### <a name="is-there-a-cap-to-how-many-microsoft-365-business-seats-a-customer-can-have"></a>Er der en lua å hvor mange Microsoft 365 firmaet sitteplass en kunden kanne ha? 
Microsoft 365 Business ble utviklet for små til mellom store bedrifter med lav til middels krav til IT-kompleksitet. Kunder kan kjøpe opptil 300 Microsoft 365 Business lisenser for organisasjonen. Kunder kan mikse og matche Sky abonnementer. som et resultat, avhengig av organisasjonens IT-krav, kan kunder legge til Microsoft 365 Enterprise-lisenser i samme konto. 
 
Når kunder vurderer et miljø som består av flere abonnementstyper, bør de samarbeide med sine betrodde IT-rådgivere for å finne ut hvordan de best kan administrere og sikre de ulike abonnementene som Microsoft 365 Business-og Microsoft 365 Enterprise-Bruk ulike funksjoner for å sikre og administrere programmer og data.

### <a name="can-i-combine-microsoft-365-business-with-other-microsoft-subscription-offerings"></a>Kan jeg kombinere Microsoft 365 Business med andre abonnementstilbud fra Microsoft? 
Ja, kunder kan kombinere Microsoft 365 Business-abonnementer med planer og tillegg fra Azure, Dynamics 365, Enterprise Mobility + Security og Office 365.

### <a name="is-everyone-in-my-business-required-to-have-a-microsoft-365-business-subscription"></a>Er alle i bedriften min pålagt å ha et Microsoft 365 Business-abonnement? 
Nei, ikke alle trenger et Microsoft 365 Business-abonnement, selv om sikkerhets-og administrasjons fordelene bare er tilgjengelige for brukere med enheter som administreres med et Microsoft 365 Business-abonnement. 
 
Standardisering et IT-miljø bidrar til å redusere vedlikeholds-og sikkerhets kostnadene over tid, og er en tilstand som bedrifter bør strebe etter å oppnå. Vi erkjenner imidlertid at noen små og mellom store kunder oppdaterer sin programvare først og fremst når de oppgraderer maskinvaren sin over en lengre periode. Bedrifter kan distribuere Microsoft 365 Business til en del av organisasjonen, men for å oppnå best mulig beskyttelse av sensitive forretningsdata og konsekvente samarbeids opplevelser, anbefales distribusjon til alle brukere.

### <a name="how-can-i-know-if-the-hardware-and-software-i-run-today-is-compatible-with-microsoft-365-business"></a>Hvordan kan jeg vite om maskinvaren og programvaren jeg kjører i dag, er kompatibel med Microsoft 365 Business? 
Hvis maskinvaren du kjører i dag, kjører Windows 7 Pro eller nyere, oppfyller det sannsynligvis minimumskravene for Microsoft 365 Business. Enkelte Windows 10-funksjoner, for eksempel Cortana, Windows Hello og multitouch, krever spesifikk maskinvare som bare er tilgjengelig på nyere PC-er. Se <a href="https://www.microsoft.com/windows/windows-10-specifications" target="_blank">systemkravene for Windows 10 Pro</a> hvis du vil ha mer informasjon. 
 
Kompatibilitet for eksisterende skrivebordsprogram (Win32) er sterk i Windows 10, og de fleste eksisterende programmer fungerer uten endringer. Kundenes og deres tiltrodd den rådgivere burde lese det anbefalt søknad tester forarbeide for <a href="https://docs.microsoft.com/windows/deployment/planning/windows-10-compatibility#recommended-application-testing-process" target="_blank">Vinduer 10 forenlighet</a> og anmelde <a href="https://products.office.com/office-system-requirements#subscription-plans-section" target="_blank">kontoret system behov</a> å sikre en glatte overgang å Microsoft 365 firmaet. 

### <a name="what-is-windows-10-business"></a>Hva er Windows 10 Business? 
Windows 10 Business er et sett med funksjoner for skytjenester og enhetsadministrasjon som utfyller Windows 10 Pro og aktiverer de sentraliserte administrasjons-og sikkerhetskontrollene til Microsoft 365 Business. Windows 10 Business leveres også med Windows AutoPilot, en tjeneste som effektiviserer distribusjonen av nye Windows 10-PCer. Hvis du har enheter som er lisensiert for Windows 7, 8 og 8,1 Professional, gir Microsoft 365 Business en oppgradering til Windows 10 Pro, som er forutsetningen for distribusjon av Windows 10 Business.
> [!IMPORTANT]
> Du nød å levere originalen fabrikat nøkkel når du oppgradere, ellers det oppgradere wont ' arbeide. 

### <a name="how-does-microsoft-365-business-help-support-our-bring-your-own-device-byod-policy"></a>Hvordan hjelper Microsoft 365 Business med å støtte våre retningslinjer for å ta med egen enhet (BYOD)? 
Mange ansatte foretrekker å bruke sine egne mobiltelefoner eller nettbrett for å få tilgang til personlig og arbeidsinformasjon i stedet for å bære flere enheter for hvert formål. Bruken av personlige enheter for arbeid, mens vanlig, øker risikoen for at forretningsinformasjon kan ende opp i gale hender. Mange konkurrerende mobildata beskyttelses løsninger krever at brukerne bytter til en bestemt modus på enheten, eller bruker en annen kompleks mekanisme som brukerne kan finne påtrengende og derfor unngå å bruke. 
 
Microsoft 365 Business tilbyr kundene en enkel, men kraftig måte å gjøre det mulig for ansatte å bruke sine personlige enheter for arbeid samtidig som det gir bedriften muligheten til å hindre disse enhetene i å få tilgang til, beholde og/eller dele forretningsinformasjon . Mer spesifikt: 
* **App Protection for Office Mobile Apps** bidrar til å beskytte Office-data, inkludert e-post, kalender, kontakter og dokumenter på IOS og Android mobile enheter, ved å håndheve policyer som automatisk sletting av forretningsdata etter en fastsatt tid for ikke å koble til tjenesten, som krever at informasjon lagres bare til OneDrive for Business, krever en PIN-/fingeravtrykks verifisering for å få tilgang til Office-programmer, og hindre at bedriftens data blir kopiert fra en Office-app til personlige programmer. Administrasjon av mobilprogrammer for andre Mobilapper via Intune er også tilgjengelig for Microsoft 365 Business-abonnenter. 
  
* **Enhetsbehandling for Windows 10-PCer** gjør det mulig for bedrifter å velge å angi og håndheve funksjoner som Windows Defender-beskyttelse for skadelig programvare, automatiske oppdateringer og deaktivering av skjermbilder etter en fastsatt tidsperiode. I tillegg kan tapte eller stjålne Windows 10-enheter bli fullstendig slettet av forretningsprogrammer og data via administrasjonssenteret. 

* **Enhetsbehandling for IOS, Android & MacOS** -funksjoner hjelper bedrifter med å administrere et mangfoldig økosystem av enheter som inkluderer IOS-, Android-, Windows-og MacOS-enheter. 

### <a name="how-does-microsoft-365-business-help-protect-pcs-in-my-organization-from-malicious-attacks"></a>Hvordan hjelper Microsoft 365 Business med å beskytte PC-er i organisasjonen mot ondsinnede angrep? 
PCer administrert med Microsoft 365 Business er beskyttet med Windows Defender, som er nei. 1 antivirus funksjon i Windows 10, som beskytter flere datamaskiner mot virus, skadelig programvare, spionprogrammer og andre trusler enn noen annen løsning.  Med Microsoft 365 Business kan bedrifter sikre at Windows Defender-beskyttelsen kjører og alltid er oppdatert på alle Windows 10-enhetene.  
 
Windows 10 Business inkluderer også Windows Defender Utnyttelses Guard, et nytt sett med funksjoner for hindring av inntrenging. En av funksjonene, styrt mappetilgang, stopper ransomware i sine spor ved å låse ned mapper og hindre uautoriserte apper fra å få tilgang til en brukers viktige filer. 

### <a name="whats-the-difference-between-office-365-business-premium-microsoft-365-business-and-microsoft-365-enterprise"></a>Hva er forskjellen mellom Office 365 Business Premium, Microsoft 365 Business og Microsoft 365 Enterprise? 
Microsoft har en rekke ulike produktivitets-og sikkerhets administrasjons tilbud som små og mellom store kunder kan vurdere når de oppgraderer skrivebordet og enhets infrastrukturen, og hver av dem har stadig kraftigere funksjoner og funksjonalitet. 

**Office 365 Business Premium** leverer klassens beste produktivitet med Office 365-apper og-tjenester, men inkluderer ikke program beskyttelse og funksjoner for Enhetsbehandling i Microsoft 365 Business.

**Microsoft 365 Business** kombinerer Office 365-apper og-tjenester med administrasjon av mobilprogrammer og Windows 10 Pro for å muliggjøre ekstern administrasjon og beskytte enheter mot virus og skadelig programvare. Det inkluderer en forenklet administrasjonskonsoll der enhets-og datapolicyer kan administreres. Mange små og mellom store bedrifter kan være best tjent med Microsoft 365 Business. 
 
**Microsoft 365 Enterprise** er et sett med lisensierings planer som tilbyr økte nivåer av samsvar og sikkerhetsadministrasjon over Microsoft 365 Business og er utformet for bedriftskunder og kunder som har over 300 brukere. I tillegg har Microsoft 365 Enterprise-abonnementer ekstra funksjonalitet, inkludert forretningsintelligens-og analyseverktøy.

Ja, kunder kan bytte planer fra en kvalifiserende Office 365-plan til Microsoft 365 Business. Avhengig av kundens nåværende plan kan det være en nedgang eller økning i månedlige kostnader.

Microsoft 365 Business vil være tilgjengelig for alle partnere og kunder der Office 365 er tilgjengelig. Se listen over <a href="https://products.office.com/business/international-availability" target="_blank">internasjonal tilgjengelighet for Office 365 for språk, land og områder</a>.

En Microsoft 365 Business prøveversjon er tilgjengelig for CSPer. En prøveordning for direkte kunder vil være tilgjengelig senere. 
 
### <a name="what-should-customers-and-partners-know-before-running-microsoft-365-business-within-their-organization"></a>Hva bør kunder og partnere vite før de kjører Microsoft 365 Business innen organisasjonen? 
Kunder som ønsker å oppleve de fullstendige funksjonene til Microsoft 365 Business, må kjøre Windows 7, 8,1 eller 10 Pro<sup>[2](#footnote2)</sup> på de eksisterende skrivebordene. Eksisterende Windows 10 Pro-PCer bør kjøre Creators Update hvis de ikke allerede har gjort det. 

## <a name="deployment"></a>Distribusjon

### <a name="what-should-customers-consider-when-planning-a-microsoft-365-business-deployment"></a>Hva bør kundene vurdere når de planlegger en Microsoft 365 Business-distribusjon? 
Den mest direkte veien til en vellykket Microsoft 365 Business-distribusjon er å engasjere seg med en Microsoft-partner. De har omfattende opplæring og erfaring med et bredt spekter av kunde scenarioer og er best rustet til å forstå dine omgivelser og behov. Kunder som har opplevd det i staben, kan bruke <a href="https://support.office.com/article/496e690b-b75d-4ff5-bf34-cc32905d0364" target="_blank">microsoft 365 Business komme i gang</a> for å hjelpe dem i Microsoft 365 Business Deployment. 

### <a name="does-microsoft-365-business-include-the-full-capabilities-of-microsoft-intune"></a>Omfatter Microsoft 365 Business alle funksjonene i Microsoft Intune? 
Ja, Microsoft 365 Business-abonnenter lisensieres til å bruke fullstendige Intune-funksjoner for iOS, Android, MacOS og annen Enhetsbehandling på tvers av plattformer. Funksjoner som ikke er tilgjengelige i forenklet administrasjon konsollen i Microsoft 365 Business, som 3dje parts app administrasjon og konfigurasjon av WiFi-profiler, VPN-sertifikater, etc, kan administreres i full Intune konsoll. 
  

### <a name="does-azure-active-directory-premium-p1-come-with-microsoft-365-business"></a>Leveres Azure Active Directory Premium P1 med Microsoft 365 Business? 
Microsoft 365 Business inkluderer utvalgte Azure AD Premium P1 (AADP P1)-funksjoner, for eksempel selvbetjent tilbakestilling av passord med AD-skriving, Azure MFA og betinget tilgang. Det inkluderer ikke helheten av AADP P1. Vennligst se [Microsoft 365 Business service Description](https://technet.microsoft.com/library/b042d1fd-c997-46d8-8c02-1df4c932f148) for mer informasjon 
 
 
### <a name="does-microsoft-365-business-allow-customers-to-manage-macs"></a>Kan Microsoft 365 Business tillate kunder å administrere Mac? 
Intune hjelper deg med å administrere iOS-, Android-, Windows-og macOS-enheter på en sikker måte. 
 
### <a name="what-is-windows-autopilot"></a>Hva er Windows AutoPilot? 
Windows AutoPilot er en tjeneste som effektiviserer distribusjonen av nye Windows 10-PCer. Denne prosessen kan gjøres når sluttbrukeren logger seg på Microsoft 365 Business for første gang – uten at det berører enheten noensinne – ved å utnytte sentralisert administrasjonskontroller for Microsoft 365 Business. Du kan også bruke Windows AutoPilot for eksisterende PC-er som kjører Windows 10 Professional Creators Update (eller senere) og har blitt fabrikk tilbakestilt. Detaljer om Windows AutoPilot finnes i <a href="https://blogs.technet.microsoft.com/windowsitpro/2017/06/29/modernizing-windows-deployment-with-windows-autopilot/" target="_blank">juni, 2017 blogginnlegg</a>.

## <a name="compatibility"></a>Kompatibilitet

### <a name="can-i-add-office-365-add-ons-to-microsoft-365-business"></a>Kan jeg legge til Office 365-tillegg i Microsoft 365 Business? 
Alle tilleggene som kan legges til i Office 365 Business Premium, kan legges til i Microsoft 365 Business. Dette betyr at du kan kjøpe avansert trusselbeskyttelse, Office 365 Cloud app Security, avansert overholdelse, Threat Intelligence, MyAnalytics, PowerBI Pro og lydkonferanser. 
 
### <a name="can-i-add-phone-system-and-calling-plans-to-microsoft-365-business"></a>Kan jeg legge til telefon system og abonnementer på Microsoft 365 Business? 
Nei, telefon system og abonnement er reservert for kunder som har mer avanserte behov. Kunder som trenger disse funksjonene, bør se på Microsoft 365 Enterprise-tilbud. 
 
### <a name="can-microsoft-365-business-customers-use-windows-defender-advanced-threat-protection"></a>Kan Microsoft 365 bedriftskunder bruke Windows Defender avansert trusselbeskyttelse? 
Nei, kunder som krever Windows Defender avansert trusselbeskyttelse, trenger enten Windows 10 Enterprise E5 eller Microsoft 365 Enterprise E5. 
 


### <a name="can-customers-create-hosted-windows-10-vms-with-a-microsoft-365-business-subscription"></a>Kan kunder opprette vertsbaserte Windows 10 VMs med et Microsoft 365 Business-abonnement? 
Nei, kunder som krever virtualisering, bør kjøpe Windows 10 Enterprise eller et Microsoft 365 Enterprise-abonnement.

### <a name="what-is-the-gdpr-and-how-does-microsoft-365-business-help-customers-with-their-compliance-obligations"></a>Hva er GDPR, og hvordan hjelper Microsoft 365 business kunder med sine samsvars forpliktelser?
EUs personvernforordning (GDPR) er en omfattende ny personvernlovgivning som gir innbyggerne i den europeiske union (EU) større kontroll over sine "personlige data" og krever at organisasjoner opprettholder integriteten til disse personopplysningene. GDPR krever at organisasjoner som kontrollerer, eller behandler personopplysninger knyttet til EU-innbyggere, bare kan bruke tredjeparts databehandlere som oppfyller GDPR-kravene for behandling av personopplysninger. I mars 2017 gjorde Microsoft tilgjengelige kontraktsmessige garantier som gir disse garantiene. Kunder som har spørsmål om hvordan Microsoft kan hjelpe dem med å oppfylle sine ytterligere GDPR-forpliktelser, bør lære om de avanserte samsvars-og sikkerhetsfunksjonene som er tilgjengelige som tilleggsprogrammer (for eksempel Azure Information Protection, hindring av tap av data, avansert Trusselbeskyttelse, osv.) og i andre suiter (for eksempel Microsoft 365 Enterprise E5).
